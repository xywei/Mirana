Mirana

A Moving Mesh Toolbox

Introduction

MIRANA stands for "Making It Really Adaptive Needs Adapters". And what this toolbox does is to offer such adapters connecting physical and computational spaces.

Dependency

  • ITSOL ( ITERATIVE SOLVERS packag).
  • pARMS (parallel Algebraic Recursive Multilevel Solvers).
