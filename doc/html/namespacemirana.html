subroutine mirana::mbc_d ( double precision, dimension(:,:), intent(inout), allocatable  x1,

double precision, dimension(:,:), intent(inout), allocatable  y1,
double precision, dimension(:,:), intent(inout), allocatable  y2 
)

(Mesh Boundary Condition for Derivatives) Set the boundary condition of the moving mesh.

Note
Be sure to call Mirana::d1 and Mirana::d2 on the mesh before calling this subroutine!
Parameters
x1,N-by-Nreal array, resulted from call d1(mx,hxi,x1).
x2,N-by-Nreal array, resulted from call d2(mx,het,x2).
y1,N-by-Nreal array, resulted from call d1(my,hxi,y1).
y2,N-by-Nreal array, resulted from call d2(my,het,y2).
  • Use uniform mesh at left/right boundary.
  • \(\xi\)-mesh is parallel to the wall at floor/ceiling.

(Non-conservative Mesh Differentiation) Compute derivatives of mesh grid with respect to computational space and inverse using non-conservative form central difference.

Parameters
mxN-by-N real matrix, x-coordinate of mesh grid (fully determined).
myN-by-N real matrix, y-coordinate of mesh grid (fully determined).
hxireal number, step size in computational domain along \(\xi\) direction.
hetreal number, step size in computational domain along \(\eta\) direction.
Returns
jcb real N-by-N matrix, the Jacobian \(J\).
xi_x real N-by-N matrix, \(\partial\xi/\partial x\).
xi_y real N-by-N matrix, \(\partial\xi/\partial y\).
et_x real N-by-N matrix, \(\partial\eta/\partial x\).
et_y real N-by-N matrix, \(\partial\eta/\partial y\).
